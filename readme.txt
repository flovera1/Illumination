Proyecto: proyecto de iluminacion

Nos toco desarrollar la iluminacion sobre el archivo 06.mb
por ello le dimos este nombre al proyecto.

Logramos cumplir con todos los requerimientos que se nos pidio
en el proyecto:
1. Luz clave, luz de rebote.
2. Luz de ambiente.
3. Luz de relleno
4. Rim light.

Ademas hicimos la parte de la neblina utilizando Fog.

Para esta ultima parte, no seguimos el video recomendado, utilizamos
un spotlight que es una herramienta que provee maya para lograr lo mismo.
Asi que en el proyecto encontrara los parametros de:
	fog spread
	fog intensity

si desea cambiarlas para ver el efecto en el rendering de la imagen.
